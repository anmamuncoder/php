| MVC          | Django      | PHP Mini MVC                     |
| ------------ | ----------- | -------------------------------- |
| Model        | `models.py` | `models/User.php`                |
| View (logic) | `views.py`  | `controllers/UserController.php` |
| Template     | `.html`     | `views/*.php`                    |
| URL          | `urls.py`   | `index.php` (router)             |
